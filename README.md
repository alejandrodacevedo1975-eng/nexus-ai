# ⚡ Vortex AI Hub

Tu plataforma personal de chat con múltiples modelos de IA gratuitos.

## Modelos incluidos (todos GRATIS)
- Llama 3.1 8B
- DeepSeek R1
- Mistral 7B
- Qwen 2.5 7B
- Gemma 3 27B
- Phi-3 Mini
- OpenChat 7B

## Cómo hacer deploy en Railway

1. Sube esta carpeta a GitHub (nuevo repositorio)
2. Ve a railway.app → New Project → Deploy from GitHub
3. Selecciona el repositorio
4. Railway detecta Node.js automáticamente
5. ¡Listo! Tu app estará en línea en 2 minutos

## Configuración
Al abrir la app por primera vez te pedirá tu API Key de OpenRouter.
Ve a openrouter.ai/settings/keys, crea una key y pégala ahí.
